# design-system/tokens

> **Design tokens layer. Defines all reusable design variables in the project.**
> This is the bottom layer of the design system, and all components reference these tokens.

## Files

| File | Role | Purpose |
|------|------|---------|
| `index.ts` | Entry | Re-exports all tokens |
| `colors.ts` | Token | Color system (primary, neutral, semantic) |
| `spacing.ts` | Token | Spacing system (4px base) |
| `typography.ts` | Token | Fonts, font sizes, line heights |
| `shadows.ts` | Token | Shadow levels |
| `radii.ts` | Token | Border radius |
| `breakpoints.ts` | Token | Responsive breakpoints |
| `transitions.ts` | Token | Animation duration and easing |
| `z-index.ts` | Token | Z-index system |
