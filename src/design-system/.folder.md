# Folder: /src/design-system

1. **"地位"**：OpenBento 的设计系统，提供设计令牌、原子组件和复合组件模式，遵循 Atomic Design 原则。
2. **"逻辑"**：Tokens → Foundation → Primitives → Patterns，自底向上构建，上层依赖下层，提供统一的视觉语言。
3. **"约束"**：所有组件必须使用设计令牌，严禁硬编码颜色和尺寸，必须支持主题切换。

## 🧩 成员清单

- `tokens/`：设计令牌（颜色、间距、阴影等）
- `foundation/`：基础设施（主题、网格）
- `primitives/`：原子组件（Box, Flex, Text, Stack）
- `patterns/`：复合组件（Button, Card, Modal 等）
- `utils/`：工具函数（cn, polymorphic）
