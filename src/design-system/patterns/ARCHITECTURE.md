# design-system/patterns

> **Composite component patterns. Reusable UI modules composed of atomic components.**
> Each component follows Component/Types/Index structure.

## Component Folders

| Folder | Component | Purpose |
|--------|-----------|---------|
| `Avatar/` | Avatar | User avatar |
| `Badge/` | Badge | Badge/label |
| `Button/` | Button | Button (multiple variants) |
| `Card/` | Card | Generic card container |
| `Dropdown/` | Dropdown | Dropdown menu |
| `Icon/` | Icon | Icon wrapper component |
| `Input/` | Input | Input field |
| `Modal/` | Modal | Modal dialog |
| `Tooltip/` | Tooltip | Tooltip |

## Files

| File | Role | Purpose |
|------|------|---------|
| `index.ts` | Entry | Re-exports all Pattern components |
