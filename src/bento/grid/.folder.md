# Folder: /src/bento/grid

1. **"地位"**：网格布局系统核心，提供 CSS Grid 容器和响应式支持，管理卡片的排列和定位。
2. **"逻辑"**：接收卡片配置和尺寸信息，通过 CSS Grid 计算布局，提供响应式断点和自动填充能力。
3. **"约束"**：必须使用 BENTO_UNIT 和 BENTO_GAP 常量，严禁硬编码尺寸值，必须支持响应式布局。

## 🧩 成员清单

- `BentoGrid.tsx`：网格容器组件，CSS Grid 实现
- `BentoGrid.types.ts`：网格相关类型定义
- `BentoCell.tsx`：网格单元格组件
- `useGridLayout.ts`：网格布局计算 Hook
