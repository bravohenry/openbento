# Folder: /src/bento/grid

1. **"Status"**: Core of the grid layout system, providing CSS Grid container and responsive support, managing card arrangement and positioning.
2. **"Logic"**: Receives card configuration and size information, calculates layout through CSS Grid, provides responsive breakpoints and auto-fill capabilities.
3. **"Constraints"**: Must use BENTO_UNIT and BENTO_GAP constants, hard-coded size values are prohibited, must support responsive layout.
