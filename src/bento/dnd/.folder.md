# Folder: /src/bento/dnd

1. **"地位"**：拖拽交互系统，提供网格内卡片的拖拽和交换能力，基于 @dnd-kit 实现。
2. **"逻辑"**：通过 GridDndProvider 提供拖拽上下文，DraggableGridItem 包装可拖拽项，基于交换逻辑实现位置变更。
3. **"约束"**：必须使用 @dnd-kit 库，严禁直接操作 DOM，必须支持网格吸附和交换动画。

## 🧩 成员清单

- `GridDndProvider.tsx`：拖拽提供者组件，基于 @dnd-kit/core
