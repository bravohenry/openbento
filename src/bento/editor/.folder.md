# Folder: /src/bento/editor

1. **"地位"**：编辑器 UI 层，提供编辑模式的界面和交互，管理 Widget 的创建、编辑、删除操作。
2. **"逻辑"**：通过 EditorContext 管理编辑状态，EditorToolbar 提供操作按钮，WidgetEditOverlay 提供编辑覆盖层，ProfileSection 显示用户信息。
3. **"约束"**：所有编辑操作必须通过 EditorContext，严禁直接修改 Widget 状态，必须支持撤销/重做。

## 🧩 成员清单

- `EditorContext.tsx`：编辑器状态管理 Context
- `EditorToolbar.tsx`：编辑器工具栏组件
- `EditorFooter.tsx`：编辑器底部栏组件
- `WidgetEditOverlay.tsx`：Widget 编辑覆盖层
- `ProfileSection.tsx`：用户信息展示组件
