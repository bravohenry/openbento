# Folder: /src/bento/editor

1. **"Status"**: Editor UI layer, providing interface and interactions for edit mode, managing Widget creation, editing, and deletion operations.
2. **"Logic"**: Manages edit state through EditorContext, provides action buttons via EditorToolbar, provides edit overlay through WidgetEditOverlay, displays user information in ProfileSection.
3. **"Constraints"**: All edit operations must go through EditorContext, direct modification of Widget state is prohibited, must support undo/redo.
