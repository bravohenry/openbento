# Folder: /src/bento/editor

1. **"Status"**: Editor UI layer, providing interface and interactions for edit mode, managing Widget creation, editing, and deletion operations. Includes layout management, toolbar actions, and profile editing.
2. **"Logic"**: Manages edit state through EditorContext, provides two-column layout via EditorLayout (left: profile section, right: widget grid), provides action buttons via EditorToolbar (widget creation, sharing, view mode toggle), provides edit overlay through WidgetEditOverlay, displays user information in ProfileSection. Supports desktop/mobile view modes with separate widget layouts.
3. **"Constraints"**: All edit operations must go through EditorContext, direct modification of Widget state is prohibited, must support desktop/mobile view mode separation.
