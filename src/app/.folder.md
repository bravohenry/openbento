# Folder: /src/app

1. **"地位"**：Next.js App Router 路由和页面层，处理路由导航和页面渲染，遵循 Next.js 约定。
2. **"逻辑"**：每个子文件夹对应一个路由，page.tsx 是页面组件，layout.tsx 提供布局，通过路由参数传递数据。
3. **"约束"**：必须遵循 Next.js App Router 约定，严禁在页面组件中直接操作 DOM，必须使用服务端组件优先。

## 🧩 成员清单

- `/`：首页（page.tsx）
- `/bento`：Bento 展示页
- `/bento/editor`：Bento 编辑器（主功能页）
- `/showcase`：案例展示
- `layout.tsx`：根布局，包含全局样式和字体
- `globals.css`：全局 CSS 和 Tailwind 配置
